# movie_app_2019

### 간단 설명
- React로 만든 영화 웹서비스입니다.
- 영화 목록을 디자인된 형식으로 보여줍니다.

### 제공 기능
- 영화 정보를 제공합니다.(영화 제목, 개봉년도, 장르, 소개글)
